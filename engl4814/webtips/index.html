Managing a Website

Document Summary

This page provides tips on managing a website. It explains the folder structure you should maintain and discusses considerations for filenames.

Managing a website, as discussed in this document, requires a web hosting account.

To manage a website, you should consider a few primary factors:

  • The folder structure of your website
  • Steps to upload your files
  • The URL to your website

Steps to Upload Files

Depending on your hosting provider, you typically use the following steps to upload files:

  1. Access your hosting provider's logon page.
  2. Log on to your account.
  3. Access the file management feature on your control panel.
  4. Select the web root folder for your website.
  5. If required, create a new subfolder to contain your pages.
  6. Use the file upload feature to upload a copy of your files to your web folder.

Website URL

For users to access your website, they must enter the URL, including any subfolders. If you use index as the filename for your landing page, users typically do not have to include the filename in the URL.

For example, if you post an index.html page in a public_html/webtips folder in your hosting account (where public_html is the name of your web root folder), a user can simply enter a URL similar to www.engl4814.com/webtips.
